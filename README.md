# Projeto de Automação de Formulário de Contato - LearnWorlds

Automação de testes utilizando **Selenium WebDriver**, **JavaScript (Node.js)** e **Mocha**,
com estrutura **Data-Driven** baseada em JSON.

Este projeto realiza o preenchimento e envio do formulário de contato na plataforma educacional
**LearnWorlds**, acessível em:  
🔗 [https://iterasys.learnworlds.com](https://iterasys.learnworlds.com)

---

## Tecnologias Utilizadas

- **Selenium WebDriver**  
- **JavaScript (Node.js)**  
- **Mocha** como framework de testes  
- **Firefox** como navegador de execução

---

## 📁 Estrutura do Projeto

```
/fixtures
  └── contactData.json
/test
  └── contact.spec.js
```

---

## Como Executar os Testes

```bash
npm install
npx mocha
```
